Radar

Java/C++

Radar Project using the Arduino IDE and Processing IDE, combining hardware and software to visualize real-time object detection. The system utilized ultrasonic sensors to detect objects within a specific range and angle, displaying the results in a radar-like interface. Key Features: Hardware Implementation: Used an Arduino microcontroller paired with an ultrasonic sensor for distance and angle measurement. A servo motor rotated the sensor to scan the area. Software Visualization: The Processing IDE created a radar-like graphical display to represent detected objects in real-time. Data from the Arduino was transmitted via serial communication for visualization. Real-Time Interaction: Displayed object positions dynamically as the sensor scanned the environment.
